# Εντολή ssh-keygen: παραδείγματα δημιουργίας SSH keys

Πρακτικά παραδείγματα ssh-keygen: SSH keys Ed25519/RSA, όνομα αρχείου, passphrase, fingerprint, GitHub και δοκιμή σύνδεσης σε server.

Η δημιουργία SSH key σημαίνει δημιουργία ζεύγους public/private key. Για νέα keys, το Ed25519 είναι συνήθως η πιο πρακτική επιλογή· το RSA 4096 χρειάζεται κυρίως για παλιά συστήματα.

## Γρήγορη απάντηση

```bash
ssh-keygen -t ed25519 -C "you@example.com"
ssh-keygen -t rsa -b 4096 -C "you@example.com"
ssh-keygen -t ed25519 -C "work@example.com" -f ~/.ssh/id_ed25519_work
ssh-keygen -lf ~/.ssh/id_ed25519.pub
```

## Βασική σύνταξη ssh-keygen

```bash
ssh-keygen -t TYPE -b BITS -C COMMENT -f OUTPUT_FILE
```

- `-t`: τύπος key
- `-b`: μέγεθος RSA
- `-C`: comment
- `-f`: αρχείο εξόδου
- `-p`: αλλαγή passphrase
- `-l`: fingerprint
- `-y`: public key από private key

## Ed25519

Για νέα keys επιλέξτε συνήθως Ed25519. Χρησιμοποιήστε RSA 4096 όταν παλιός SSH server ή εργαλείο δεν υποστηρίζει Ed25519.

```bash
ssh-keygen -t ed25519 -C "you@example.com"
```

## RSA 4096

Για νέα keys επιλέξτε συνήθως Ed25519. Χρησιμοποιήστε RSA 4096 όταν παλιός SSH server ή εργαλείο δεν υποστηρίζει Ed25519.

```bash
ssh-keygen -t rsa -b 4096 -C "you@example.com"
```

## Όνομα αρχείου, fingerprint και ssh-agent

Για τις περισσότερες συνδέσεις αρκεί ssh user@host. Χρησιμοποιήστε -i για συγκεκριμένο key, -p για διαφορετικό port και -J για σύνδεση μέσω bastion host.

```bash
ssh-keygen -t ed25519 -C "work@example.com" -f ~/.ssh/id_ed25519_work
ssh-keygen -p -f ~/.ssh/id_ed25519
ssh-keygen -lf ~/.ssh/id_ed25519.pub
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

## Προσθήκη του key σε GitHub, GitLab ή server

Η συνηθισμένη ροή: δημιουργείτε key, αντιγράφετε το public key, το προσθέτετε σε GitHub/GitLab ή server, φορτώνετε το private key στο ssh-agent και δοκιμάζετε τη σύνδεση.

```bash
ssh -T git@github.com
ssh -v user@server.example.com
```
