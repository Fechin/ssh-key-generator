# ssh-keygen command: SSH key बनाने के उदाहरण

ssh-keygen के practical examples: Ed25519/RSA SSH key, file name, comment, passphrase, fingerprint, GitHub और server login test।

SSH key generate करने का मतलब public key और private key की pair बनाना है। नई key के लिए आम तौर पर Ed25519 बेहतर है; पुराने server या tool की जरूरत हो तो RSA 4096 इस्तेमाल करें।

## त्वरित उत्तर

```bash
ssh-keygen -t ed25519 -C "you@example.com"
ssh-keygen -t rsa -b 4096 -C "you@example.com"
ssh-keygen -t ed25519 -C "work@example.com" -f ~/.ssh/id_ed25519_work
ssh-keygen -lf ~/.ssh/id_ed25519.pub
```

## ssh-keygen basic syntax

```bash
ssh-keygen -t TYPE -b BITS -C COMMENT -f OUTPUT_FILE
```

- `-t`: key type
- `-b`: RSA bit size
- `-C`: comment
- `-f`: output file
- `-p`: passphrase change
- `-l`: fingerprint
- `-y`: private key से public key निकालना

## Ed25519

नई keys के लिए आम तौर पर Ed25519 चुनें। पुराना SSH server या tool Ed25519 support न करे तो RSA 4096 इस्तेमाल करें।

```bash
ssh-keygen -t ed25519 -C "you@example.com"
```

## RSA 4096

नई keys के लिए आम तौर पर Ed25519 चुनें। पुराना SSH server या tool Ed25519 support न करे तो RSA 4096 इस्तेमाल करें।

```bash
ssh-keygen -t rsa -b 4096 -C "you@example.com"
```

## File name, fingerprint और ssh-agent

ज्यादातर connections के लिए ssh user@host काफी है। Specific key के लिए -i, non-default port के लिए -p और bastion host के लिए -J इस्तेमाल करें।

```bash
ssh-keygen -t ed25519 -C "work@example.com" -f ~/.ssh/id_ed25519_work
ssh-keygen -p -f ~/.ssh/id_ed25519
ssh-keygen -lf ~/.ssh/id_ed25519.pub
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

## GitHub, GitLab या server में key add करें

Normal flow: key generate करें, public key copy करें, GitHub/GitLab या server में add करें, private key ssh-agent में load करें और connection test करें।

```bash
ssh -T git@github.com
ssh -v user@server.example.com
```
