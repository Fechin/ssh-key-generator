# ssh-keygen 명령어: SSH 키 생성 예제

ssh-keygen으로 Ed25519/RSA SSH 키를 만들고 파일명, 주석, 패스프레이즈, 지문, GitHub 및 서버 접속을 확인하는 예제입니다.

SSH 키를 생성한다는 것은 공개키와 개인키 한 쌍을 만드는 것입니다. 새 키라면 Ed25519가 가장 무난하고, 오래된 시스템과의 호환성이 필요할 때 RSA 4096비트를 선택합니다.

## 빠른 답변

```bash
ssh-keygen -t ed25519 -C "you@example.com"
ssh-keygen -t rsa -b 4096 -C "you@example.com"
ssh-keygen -t ed25519 -C "work@example.com" -f ~/.ssh/id_ed25519_work
ssh-keygen -lf ~/.ssh/id_ed25519.pub
```

## ssh-keygen 기본 문법

```bash
ssh-keygen -t TYPE -b BITS -C COMMENT -f OUTPUT_FILE
```

- `-t`: 키 종류
- `-b`: RSA 비트 수
- `-C`: 주석
- `-f`: 저장 파일
- `-p`: 패스프레이즈 변경
- `-l`: 지문
- `-y`: 개인키에서 공개키 생성

## Ed25519

새 키는 대부분 Ed25519가 적합합니다. 오래된 SSH 서버나 도구가 Ed25519를 지원하지 않을 때만 RSA 4096비트를 사용하세요.

```bash
ssh-keygen -t ed25519 -C "you@example.com"
```

## RSA 4096

새 키는 대부분 Ed25519가 적합합니다. 오래된 SSH 서버나 도구가 Ed25519를 지원하지 않을 때만 RSA 4096비트를 사용하세요.

```bash
ssh-keygen -t rsa -b 4096 -C "you@example.com"
```

## 파일명, 지문, ssh-agent

대부분의 접속은 ssh user@host로 충분합니다. 특정 키는 -i, 다른 포트는 -p, 배스천 호스트는 -J를 사용합니다.

```bash
ssh-keygen -t ed25519 -C "work@example.com" -f ~/.ssh/id_ed25519_work
ssh-keygen -p -f ~/.ssh/id_ed25519
ssh-keygen -lf ~/.ssh/id_ed25519.pub
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

## GitHub, GitLab, 서버에 등록하기

기본 흐름은 키 생성, 공개키 복사, GitHub/GitLab 또는 서버 등록, ssh-agent 로드, 연결 테스트입니다.

```bash
ssh -T git@github.com
ssh -v user@server.example.com
```
